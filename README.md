# Scientific Calculator (Java GUI)

A desktop-based scientific calculator built using Java Swing.
This application supports basic arithmetic operations along with advanced scientific functions.

## Features

* Basic operations: +, -, *, /
* Scientific functions: sin, cos, tan
* Square root and logarithm calculations
* Memory operations (MC, MR, M+)
* Interactive GUI using Java Swing

## Technologies Used

* Java
* Swing (GUI)
* AWT Event Handling

## How to Run

### Option 1: Using JAR

java -jar calculator.jar

### Option 2: Using Source Code

javac ScientificCalculator.java

java ScientificCalculator
